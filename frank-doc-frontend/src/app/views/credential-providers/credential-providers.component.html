Credential Providers

Credential providers are used to provide credentials for the Frank!Framework application inorder to connect to external systems, such as a database, a web service, or any other system that requires authentication. These are commonly known as authentication aliases or `authAlias`. You can find the configured auth aliases in the FF! Console on the security items page. An authentication alias can be used to configure passwords without hardcoding them in the application. This also allows you to change passwords conditionally, when using an authentication store, such as Kubernetes Secrets, Delinea Secret Server, WildFly Credentials or one of the other available Credential Providers.
